## 🧠 技术方案

### 问题
Windows 10 Mobile 没有 ONNX Runtime 和原生 Windows ML API。
最先进平台也有 10 年历史，流通机器多为骁龙 800/801。
现代 embedding 模型性能开销太大。

### 解决方案
**手搓 DirectX 11 Compute Runtime**
- 用 C++/CX 调用 W10M 设备的 GPU 进行 DirectCompute 推理
- Adreno 330 (骁龙 800) 约 120+ GFlops
- 80M 参数量甜点规模下，单次推理 <500ms

### 模型
- 基于 **MiniGL** 架构
- 蒸馏 Google **EmbeddingGemma4-266B**
- 蒸馏过程全部本地完成

4. Emotional Inertia（50轮）— 情绪事件衰减 + 情绪残留

### 使用条件
